# Digital-Literacy-Project

## Project Overview
This is a conceptual, non-software project focused on understanding and demonstrating **digital literacy** for first-year engineering students. It does not use any programming language or tech stack. The goal is to explore what digital literacy means, why it is important, and how students can build a professional and safe digital presence.  

The work is organised into five tasks covering awareness, portfolio building, coding and collaboration tools, communication etiquette, and cybercrime awareness.

## Project Goals
- Understand the concept and importance of digital literacy.  
- Build a basic student digital portfolio on key platforms (e.g., GitHub, LinkedIn). [web:21][web:22]  
- Explore online coding and collaboration tools for academic purposes. 
- Practise professional email etiquette and responsible social media usage. 
- Study one type of cybercrime and prepare practical safety tips and reporting options for students in India. 

## Tasks Included

### Task 1 – Digital Literacy Infographic
- One-page infographic explaining digital literacy, useful tools for students, and safe internet practices.  
- Created using a free online design tool and saved as an image.   

### Task 2 – Student Digital Portfolio
- Setup or improvement of profiles on platforms like GitHub, LinkedIn, and others as applicable.  
- Basic GitHub profile README and filled Education details on LinkedIn.   

### Task 3 – Coding & Collaboration Platforms
- Solved at least one beginner-level problem on an online coding platform (e.g., HackerRank / CodeChef / HackerEarth).  
- Created a “Digital Literacy Awareness Quiz” using Google Forms and viewed responses in Google Sheets. 

### Task 4 – Email Etiquette & Social Media
- Drafted two sample professional emails:  
  - To a professor (assignment extension request).  
  - To an internship coordinator (expressing interest).  
- Prepared a simple social media Do’s and Don’ts checklist for college students. 

### Task 5 – Cybercrime Awareness
- Researched one cybercrime type (e.g., phishing, UPI fraud, identity theft, cyberbullying, or ransomware) and wrote a short case study.  
- Created a “Stay Safe Online” checklist, including UPI safety tips and details of the National Cyber Crime Portal and helpline 1930. 

## Repository Contents
- `report/` – Project report (`.docx` / `.pdf`) covering all five tasks. 
- `task-1-presentation/` – Infographic or slide image for digital literacy awareness.  
- `task-2-portfolio/` – Screenshots of professional profiles and short notes.  
- `task-3-platforms/` – Coding challenge screenshot and Google Form/Sheet screenshots.  
- `task-4-email-etiquette/` – Email drafts and social media checklist.  
- `task-5-cybercrime/` – Cybercrime case study and prevention checklist.
